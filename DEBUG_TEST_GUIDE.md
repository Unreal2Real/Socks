## 🐛 常见问题排查

### 问题 1：HTTP 500 错误

**现象：**
```
[14:06:40] ✗ 扫描失败: HTTP 500
```

**排查步骤：**
1. 检查服务器日志：
```bash
# 查看终端输出，寻找错误信息
```

2. 常见原因：
- `RuntimeError: Working outside of request context` → 已修复
- 编码错误 `'utf-8' codec can't decode` → 数据文件编码问题

### 问题 2：扫描进度不更新

**现象：**
- 扫描开始后进度长时间不变化

**排查步骤：**
1. 检查网络连接
2. 检查浏览器控制台是否有错误
3. 检查服务器日志是否有异常

### 问题 3：K线图加载失败

**现象：**
```
K线图 加载失败：network error
```

**排查步骤：**
1. 检查 `/api/stock/daily` 接口是否正常
2. 检查 `/api/stock/pattern` 接口是否正常
3. 检查浏览器控制台的具体错误

### 问题 4：股票列表为空

**现象：**
- 返回的股票列表为空数组

**排查步骤：**
1. 检查数据源配置
2. 检查网络连接（Baostock 需要联网）
3. 检查通达信路径配置（`C:/zd_zsone`）
